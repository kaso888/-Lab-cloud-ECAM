# Lab - Partie 2 - IaaS sur OVH Cloud

## Installation de Terraform sous Windows
- Télécharger puis extraire https://releases.hashicorp.com/terraform/1.3.6/terraform_1.3.6_windows_386.zip
- Editer les variables d'environnement partie system
- Sélectionner la variable puis ajouter le chemin d'installation de terraform

## Tester l'installation de Terraform
- Ouvrir un invite de commande puis lancer la commande terraform --version

## Initialisation du workspace
- Ouvrir un invite de commande
- Se placer dans le repertoire du LAB puis lancer la commande suivante terraform workspace new mon_projet_ovh
- Lancer la commande terraform init
